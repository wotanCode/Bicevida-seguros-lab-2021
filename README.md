# Bicevida Seguros

webApp que conecta con endpoints de Bicevida para obtener información de sus seguros

### Librerías utilizadas
- React versión 17.0.2
- Bootstrap versión 5.1.3

### Instrucciones.
Ejecutar los siguientes comandos en este orden.
```
npm install
npm start
```
